# Nuxt.js + Express

> Nuxt.js project using Spotify API.

## Build Setup

``` bash
# Dependencies
$ yarn install

# HMR Serve at at localhost:3000
$ npm run dev

# Production build
$ npm run build
$ npm start

# Generate static project
$ npm run generate
```
